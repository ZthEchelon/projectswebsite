HomeServer Setup · Systems / DevOps / Security

Reachable under CGNAT, hardened against compromise

The home lab stays accessible without opening router ports. Hardened SSH/SFTP, tunnels, monitoring, and backup drills deliver enterprise-grade confidence.

Constraints

  • Dynamic IP / CGNAT made port-forwarding volatile.
  • SSH/SFTP exposed to brute-force attacks.
  • Need for reliable backups + observability.

Built it

  • Key-only SSH, disabled passwords + root, strict session limits.
  • Chrooted sftpusers group with jailed directories.
  • Cloudflare Tunnel + Tailscale/WireGuard to keep traffic private.
  • Docker Compose for predictable deployments and isolation.
  • Restic/Borg backups with B2/S3 targets and restore validation.

Monitoring & reliability

  • Uptime probes for tunnel + admin UI.
  • SMART scans for disk health + alert routing.
  • Central log collection for auth events and restarts.

Takeaways

  • Misconfiguration is the primary risk—defaults lean least-privileged.
  • Tunnels win when CGNAT prevents stable inbound routes.
  • Backups only matter when restores are tested.

MindSky · Visual thinking

Graph snapshots that make experimentation feel safe

Nodes are presentational, the canvas owns behavior, and every graph state is stored as JSONB so undo/redo, autosave, and collaboration stay reliable.

Problems

  • Linear note-taking fails for exploratory thought.
  • Users feared losing work when experimenting.
  • State needed to remain visual, versionable, and shareable.

Solution

  • Canvas stores graph logic, nodes purely render themselves.
  • React Flow + custom node components for visual cues.
  • Graph snapshots committed on meaningful actions and saved via debounced autosave.
  • Backend keeps JSONB nodes/edges for atomic restores and replay.
  • Snapshot history feeds undo (Cmd+Z) and redo (Ctrl+Shift+Z / Ctrl+Y).

Snapshots capture every committed action; drag/selection changes remain transient to avoid polluting history. Autosave pauses during undo/redo so the backend mirrors the brain's stable state.

Tech

React · TypeScript · React Flow · Node.js · PostgreSQL · JSONB graph storage

Outcome: non-linear thinking feels fluid, experimentation is encouraged, and the architecture is ready for collaboration, exports, and version history.

Next: surface version history UI, multi-user collaboration, shareable export flows.
